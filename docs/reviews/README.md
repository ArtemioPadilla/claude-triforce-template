# Code Reviews & Audit Reports

This directory contains reviews and audits created by the QA agent (Centinela).

## File Types
- `{feature}-review.md` — Code review for a specific feature
- `security-audit-{date}.md` — Security audit report
- `code-health-{date}.md` — Code health scan results
- `release-check-{version}.md` — Pre-release verification
